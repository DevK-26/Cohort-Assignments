# Mintlify Documentation Website - Desktop Recreation

A desktop-first documentation website inspired by Mintlify's design, built with pure HTML and CSS.

##  Project Overview

This project recreates the Mintlify documentation website with a focus on content structure, readability, and layout accuracy. The implementation is desktop-only and uses no JavaScript, frameworks, or libraries.

##  Features & Sections

### 1. **Top Navigation Bar**
- Mintlify logo with brand colors
- Navigation links (Resources, Documentation, Customers, Blog, Pricing)
- Primary CTA buttons (Contact sales, Start for free)

### 2. **Hero Section**
- Eye-catching headline: "The Intelligent Knowledge Platform"
- Subtitle with value proposition
- Email capture form with CTA button
- Cloud-inspired background with gradient effects
- "NEW" badge for product announcements

### 3. **Documentation Preview Section**
- Three-panel layout (Sidebar, Main Content, TOC)
- Interactive sidebar navigation with tabs (Guides, API Reference, Changelog)
- Content cards for quick navigation
- Table of contents for page navigation

### 4. **Trusted By / Logos**
- Row of company logos (Stripe, Anthropic, OpenAI, Vercel, Supabase)
- Grayscale hover effect

### 5. **Feature Highlights**
- Two-column alternating layouts
- Visual mockups with browser-style frames
- Feature lists with checkmarks
- "Beautiful documentation that converts" section
- "AI-native documentation platform" section

### 6. **Intelligent Assistant / UI Preview**
- Large UI mockup showcasing chat interface
- AI assistant conversation preview
- Browser-style frame design

### 7. **Enterprise Features Section**
- Dark background theme (#111827)
- 3-column grid layout
- 6 feature cards (Security, Performance, Compliance, Collaboration, Analytics, Support)
- Icon-based visual hierarchy

### 8. **Case Studies / Customer Stories**
- 3-card grid layout
- Company-specific case studies (Anthropic, Stripe, Supabase)
- Image placeholders with emoji icons
- "Read case study" links

### 9. **Final Call-To-Action**
- Gradient background (teal to green)
- Large heading with impact
- Dual CTA buttons (Start for free, Talk to sales)

### 10. **Footer**
- 5-column layout (Brand, Product, Company, Resources, Legal)
- Social media icons (Facebook, Twitter, GitHub, LinkedIn)
- Copyright and legal information
