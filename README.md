# Cursor Landing Page - Clone Project

A desktop-first recreation of the Cursor AI Code Editor landing page, built with pure HTML and CSS (no frameworks, no JavaScript).

## üéØ Project Overview

This project recreates the Cursor website landing page with a focus on visual and structural accuracy. The goal was to match the original design as closely as possible in terms of layout, typography, colors, and overall visual hierarchy.



## üìê Sections Recreated

All 11 sections from the original Cursor website have been implemented:

### 1. **Top Navigation Bar**
- Fixed navbar with dark background
- Logo with custom SVG icon
- Navigation links (Features, Pricing, Enterprise, Docs, Changelog, Blog)
- Primary CTA buttons (Sign In, Download)
- Backdrop blur effect

### 2. **Hero Section**
- Large headline with gradient text effect
- Descriptive tagline
- Primary and secondary CTAs
- Interactive badge showing latest feature
- Full code editor mockup built entirely with CSS
  - Syntax-highlighted code
  - File explorer sidebar
  - AI chat panel
  - Window chrome with traffic lights

### 3. **Trusted By / Logos**
- Row of company logos (OpenAI, Shopify, Stripe, Samsung, Midjourney, Perplexity)
- SVG icons for brand consistency

### 4. **Feature Sections (3 Blocks)**
Three alternating two-column layouts showcasing key features:
- **Tab Autocomplete**: Predictive code editing
- **AI Chat**: Natural language codebase interaction
- **Agent Mode**: Autonomous feature building

### 5. **Feature Cards Section**
- Grid of 3-4 feature cards
- Icon-based card design
- Hover effects

### 6. **Testimonials**
- Quote cards with author information
- Avatar placeholders
- Clean card-based design

### 7. **Use Cases / Stories**
- Image + text cards
- Terminal-style visual elements
- Category tags

### 8. **Changelog / Updates**
- Chronological list of updates
- Version badges
- Date formatting

### 9. **Team / About**
- Large team grid visualization
- Company description
- Call-to-action button

### 10. **Final CTA**
- Large heading
- Single prominent button
- Gradient background effect

### 11. **Footer**
- Multi-column link structure (5 columns)
- Brand section with tagline
- Social media links
- Copyright information
